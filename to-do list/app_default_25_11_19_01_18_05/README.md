# Next.js Tailwind CSS MySQL To-Do Application

## Project Description

This is a full-stack To-Do list application built with Next.js, React, Tailwind CSS, and MySQL. It demonstrates CRUD (Create, Read, Update, Delete) operations with a MySQL database via API routes in Next.js. The application features a responsive UI styled with Tailwind CSS and supports managing To-Do items with title and completion status.

## Features

- Create new To-Do items
- View all To-Do items
- Update To-Do items (title and completion status)
- Delete To-Do items
- Responsive UI with Tailwind CSS
- API routes with Next.js for backend logic
- MySQL database integration using `mysql2/promise`
- Environment variables for secure configuration
- Strict React mode and ESLint for code quality

## Prerequisites

- Node.js (v18 or newer recommended)
- npm (comes with Node.js)
- MySQL Server (v5.7+ recommended)
- Basic knowledge of terminal/command line usage
